## ðŸ§  Approach
- Flights are grouped into **short-haul** and **long-haul**
- Lounge eligibility is estimated using **assumption-based modeling**

- Short-haul routes are assumed to have:
  - Lower premium concentration

This approach prioritizes **clarity, justification, and operational usability**.
